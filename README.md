# Python Data Processing Tool

This repository contains a simple Python script for loading, processing, and visualizing technical datasets.  
It demonstrates basic data analysis skills and documentation practices relevant for application and data development roles.

## Features
- Load CSV datasets  
- Generate summary statistics  
- Compute correlations  
- Plot data values using Matplotlib  
- Save output as `plot.png`

## Files
- **main.py** – main analysis script  
- **data.csv** – sample dataset  
- **requirements.txt** – required Python libraries  
- **plot.png** – generated output (created after running the script)
